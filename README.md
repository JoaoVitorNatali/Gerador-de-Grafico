# Gerador de Grafico

O objetivo desse programa é gerar um gráfico de forma dinâmica, conforme o usuário edita a tabela disposta no site.

# Frameworks utilizados

<ul>
  <li>Bootstrap</li>
  <li>jQuery</li>
  <li>Font Awesome</li>
  <li>Chart.js</li>
</ul>
